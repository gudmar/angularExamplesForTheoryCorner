Lifecycle methods testing

Both: child and parent components do have all lifecycle methods implemented to log their call to the pane or the console. The lifecycle methods in Angular are:

  • ngOnChanges
  • ngOnInit
  • ngDoCheck
  • ngDestroy
  • ngAfterContentInit
  • ngAfterChontentChecked
  • ngAfterViewInit
  • ngAfterViewChecked

Please note, that clicking any not related to parent or child element on the site will trigger some of this components lifecycle methods: ngDoCheck, ngAfterChontentChecked, ngAfterViewChecked. You may notice how great number of times some methods are called.

There is a ExpressionChangedAfterItHasBeenCheckedError in the console. This is expected, as to show this example there were all the lifecycle methods used to modify the page content. This is normally not allowed.
